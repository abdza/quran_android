# Tadabbur

A Quran app for Android focused on word-by-word learning and understanding, built on top of the excellent [Quran for Android](https://github.com/quran/quran_android) project.

## What is Tadabbur?

Tadabbur is a fork of [Quran for Android](https://github.com/quran/quran_android) with additional features to help users understand the Quran more deeply through word-by-word analysis.

### New Features in Tadabbur

- **Word-by-Word Display** - See the translation and transliteration of each word directly on the Quran page
- **Root Word Details** - Explore the root words and their meanings for deeper linguistic understanding
- **Enhanced Navigation** - Improved ayat navigation with nicer previous/next ayat buttons

## Setup

### Command Line

You can build Tadabbur from the command line by running `./gradlew assembleTadabburDebug`.

### Android Studio / IntelliJ

Choose "Import Project," and choose the `build.gradle` file from the top level directory. Under "Build Variants" (a tab on the left side), choose "tadabburDebug."
